# E-Commerce Application using ASP.NET Core MVC, Entity Framework Core and ASP.NET Identity (.NET 10)

## Overview

This project is part of my software development portfolio and demonstrates my ability to build a modern, database-driven web application using **ASP.NET Core MVC**, **Entity Framework Core**, and **ASP.NET Identity**.

The application is a functional e-commerce website that implements industry-standard development practices, including the MVC architectural pattern, secure user authentication, database management, and responsive web design.

The primary goal of this project was to strengthen my understanding of full-stack .NET development by building a complete application from the ground up.

## Technologies Used

* ASP.NET Core MVC (.NET 8)
* C#
* Entity Framework Core
* SQL Server
* ASP.NET Core Identity
* Razor Views
* Bootstrap 5
* HTML5
* CSS3
* JavaScript
* LINQ
* Dependency Injection

## Features

* User registration and login
* Secure authentication with ASP.NET Identity
* Role-based authorization
* Product catalogue
* Product categories
* Shopping cart functionality
* Product management (Create, Read, Update, Delete)
* Database integration using Entity Framework Core
* Responsive user interface using Bootstrap
* Server-side validation
* Error handling and user-friendly navigation
